# About pet-progect 
This service aim to achieve the complex task of organising your progects, aims, tasks, notes and other. It should simplifize your everyday routine thanks to containing each part of your progress in the one place and can be visualised immedately in the service.

# Special Settings
Use this command to correct visualising in PowerShell terminal:

```powershell
reg add HKCU\Console /v VirtualTerminalLevel /t REG_DWORD /d 1
```
